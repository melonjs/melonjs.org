<constant> KEYDOWN :String

Channel Constant for pressing a binded key
Data passed : {String} user-defined action, {Number} keyCode, {Boolean} edge state
Edge-state is for detecting "locked" key bindings. When a locked key is pressed and held, the first event will have have the third argument set true. subsequent events will continue firing with the third argument set false.
Source:
  • vendors/minpubsub.src.js, line 112
Example
me.input.bindKey(me.input.KEY.X, "jump", true); // Edge-triggered
me.input.bindKey(me.input.KEY.Z, "shoot"); // Level-triggered
me.event.subscribe(me.event.KEYDOWN, function (action, keyCode, edge) {
	 // Checking bound keys
	 if (action === "jump") {
		 if (edge) {
			 this.doJump();
		 }

		 // Make character fall slower when holding the jump key
		 this.vel.y = this.gravity;
	 }
});
